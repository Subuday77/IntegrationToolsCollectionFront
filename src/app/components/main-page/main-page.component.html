Welcome!

Below you will find a list of integration tools and useful links. Some of them will need a Java version 11 to be installed on your machine. You may check if you have a Java by executing command java -version in Command Prompt. After that you may see next message: 'java' is not recognized as an internal or external command
It means Java needs to be installed. Click here to install it automatically. If it will not work - google how to install JDK to your computer.

Tools

Tool Description Related downloads
JSON Test Tool Tool allows to check operator's responses and error handling. Validation set is a flow-chart of tests and validations. JSON test recommended validations set
Operator's JSON New tool made for API checks. May be used by operators. Validation set is also relevant for it.
Log Parser Tool allows to parse the log and to get the list of Transaction IDs and Timestamps related to them or operators login information. You may want to download the local server for this tool for faster work. Local server
Back Office Tool Set Tools are allowing to restart integration tables automatically and to create new operators on the integration server. Run from your local machine. You need to download and run Local Server to use the tools.
Request Checker Tool allows to check the requests outgoing from server. Also may be used for hash-related issues. None
Easy Link Builder Tool allows to create links for last 10 server add-on. None
Stress Test Pattern Tool allows to calculate the results of stress test. Should be used together with Log Parser. None
